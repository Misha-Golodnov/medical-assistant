# medical-assistant

## О проекте

Medical Assistant API - это сервис поддержки принятия врачебных решений, который предоставляет структурированную информацию из клинических рекомендаций Минздрава РФ на основе предполагаемого диагноза.

Врач указывает код диагноза по МКБ-10, а API возвращает:
-  Характерные симптомы заболевания
-  Рекомендуемые обследования и анализы
-  Лекарственные назначения
-  Дополнительные рекомендации по лечению

## Возможности

- **RAG-архитектура**: Точный поиск в клинических рекомендациях
- **FastAPI**: Современное и быстрое API с автоматической документацией
- **Векторный поиск**: Semantic search по медицинским документам
- **Поддержка МКБ-10**: Работа с международной классификацией болезней

## Технологический стек

- **Backend**: FastAPI, Python 3.9+
- **ML/LLM**: Sentence Transformers, OpenAI API/Llama 3
- **Векторная БД**: Chroma
- **DevOps**: Docker, GitHub Actions, pytest, flake8
